1. MO FILE DU LIEU / HE THONG PROJECT (.MPQ)
=============================================

  MO FILE DON LE (cu):
    - File > Open  hoac  Ctrl+O
    - Ho tro: .csv, .xlsx, .xls
    - CSV: se hoi ky tu phan cach (;  hoac  ,  hoac Tab) va dau thap phan
    - Excel nhieu sheet: se hien hop chon sheet

  HE THONG PROJECT (.MPQ) - QUAN TRONG:
    Mot project (.mpq) chua NHIEU QUERY chay theo thu tu.
    Moi query la 1 luong xu ly rieng co the tham chieu nhau.

    Tao project moi:    File > New Project  (Ctrl+N)
    Mo project:         File > Open Project  (Ctrl+O)
    Luu project:        File > Save Project  (Ctrl+S)
    Them query tu file: File > Add Query from File  (Ctrl+Shift+O)
                        hoac nut "+ Add Query" tren sidebar trai

  CAC LOAI QUERY:
    - From File     : load truc tiep tu CSV/Excel (bieu tuong trang)
    - Merge Queries : JOIN 2 query (left / inner / right / outer)
    - Append Queries: noi doc nhieu query thanh 1 bang (UNION)
    - From Query    : dung ket qua cua query khac lam source

  SIDEBAR QUERY (ben trai):
    - Click ten query de chuyen sang query do
    - Chuot phai: Refresh / Export / Rename / Delete
    - "+" o duoi de them query moi


2. CAC THAO TAC CO BAN
======================

  FILTER (loc dong)
    Menu: Transform > Filter Rows
    - Chon cot, chon dieu kien, nhap gia tri
    - Cot da filter se hien mui ten xanh + cham tren header
    - Dieu kien so: greater_than / less_than / ... (nhap so dung dinh dang)

  DROP COLUMNS (xoa cot)
    Menu: Transform > Drop Columns
    - Tich chon cac cot can xoa > Drop Selected
    - Cach nhanh: Ctrl+click nhieu cot tren header > chuot phai > Drop Selected

  DROP DUPLICATES (xoa trung lap chinh xac)
    Menu: Transform > Drop Duplicates
    - Xoa dong trung lap hoan toan (100% giong nau)
    - Xem AI Fuzzy Dedup (muc 12) de xoa trung lap "gan giong"

  SORT (sap xep)
    Menu: Transform > Sort Rows
    - Chon cot va chieu (Tang dan / Giam dan)

  FILL NA (dien gia tri trong)
    Menu: Transform > Fill NA
    - Chon cot cu the hoac "(all columns)"
    - Nhap gia tri de dien vao o trong (vi du: 0, N/A, Khong co)

  RENAME COLUMN (doi ten cot)
    Menu: Transform > Rename Column
    - Cach nhanh: chuot phai vao header cot > Rename

  REMOVE TOP ROWS (xoa dong dau)
    Menu: Transform > Remove Top Rows
    - Nhap so dong muon bo tu tren xuong

  USE FIRST ROW AS HEADER (dong 1 lam ten cot)
    Menu: Transform > Use First Row as Header
    - Dung khi file co dong tieu de nam trong data, khong phai header that su

  ADD INDEX COLUMN (them cot STT)
    - Them cot "STT" tu 1 den N vao vi tri dau tien
    - Chuot phai header > Add Index Column (STT)

3. GROUP ROWS (NHOM VA TONG HOP DU LIEU)
=========================================

  Tuong tu GROUP BY trong SQL / "Group By" trong Power Query.

  CACH DUNG:
    Buoc 1: Menu Transform > Group Rows...  (hoac nut "Group")
    Buoc 2: Phan "Group by" - chon cot(s) de NHOM
    Buoc 3: Phan "Aggregate" - chon ham cho tung cot con lai:
              -- (bo qua)  khong giu cot nay
              Count        dem so dong trong nhom
              Sum          cong cac gia tri so
              Mean         tinh trung binh cong
              Min / Max    gia tri nho nhat / lon nhat
              First / Last gia tri dau / cuoi trong nhom
    Buoc 4: Bam "Apply Group"

  LUU Y:
    - Phai chon it nhat 1 cot Group by VA 1 cot Aggregate
    - Sum/Mean/Min/Max nen dung voi cot so lieu
    - Count dung duoc cho moi loai cot

4. FLATTEN HIERARCHY (DU LIEU PHAN CAP - DANG WIDE)
=====================================================

  DUNG KHI NAO:
    File Excel co merged cells - moi cap phan cap co COT RIENG nhung o tren bi hop:

      Nhom hang   | Ma hang | So luong      <- cot rieng moi cap
      ------------|---------|----------
      VAT LIEU    | (trong) | (trong)       <- dong cha (merged cell)
      (trong)     | VL001   | 100           <- dong con
      (trong)     | VL002   | 200
      NHAN CONG   | (trong) | (trong)
      (trong)     | NC001   | 10

    Pandas doc vao: merged cells -> NaN o cac dong con.
    => Flatten: ffill "Nhom hang" xuong, xoa dong cha (cot con toan trong)

  CACH DUNG:
    Buoc 1: Menu Transform > Flatten Hierarchy...  (hoac nut "Flatten")
    Buoc 2: Tich chon cac cot nhom (group columns)
            - Phan mem tu dong tich cot co nhieu o trong (20-90% empty) - danh dau (*)
            - Co the chinh lai bang tay
    Buoc 3: Giu nguyen "Xoa dong cha" (mac dinh bat)
    Buoc 4: Bam "Preview" kiem tra ket qua
            - Cot nhom duoc to mau xanh duong nhat
            - Hien so dong truoc / sau
    Buoc 5: Bam "Apply"

  KET QUA:
    Nhom hang   | Ma hang | So luong
    VAT LIEU    | VL001   | 100
    VAT LIEU    | VL002   | 200
    NHAN CONG   | NC001   | 10

  PHAN BIET VOI EXPAND HIERARCHY:
    Flatten = nhieu cot rieng moi cap, cot bi trong (NaN) -> ffill xuong
    Expand  = nhieu cap CHUNG 1 cot, phan biet bang prefix (xem muc 5)

5. EXPAND HIERARCHY (DU LIEU PHAN CAP - DANG NARROW/STACKED)
==============================================================

  DUNG KHI NAO:
    ERP / phan mem ke toan export ra file ma nhieu cap phan cap
    nam CHUNG TRONG 1 COT, phan biet bang noi dung (prefix):

      Ngay viet        | SCT-Nhap      | Dien giai
      -----------------|---------------|----------
      Kho: 750-EVT-... | (trong)       | (trong)  <- cap 1: Kho
      Vat tu: 1.01...  | (trong)       | (trong)  <- cap 2: Vat tu
      2025-01-07       | 01.750.01.001 | Nhap kho <- dong du lieu that
      2025-01-08       | (trong)       | Dieu chinh

    => Can them cot "Kho" va cot "Vat tu" vao moi dong du lieu.

  CACH DUNG:
    Buoc 1: Menu Transform > Expand Hierarchy...  (hoac nut "Expand")

    Buoc 2: Chon "Cot nguon" (cot chua ca parent lau leaf) - vi du: "Ngay viet"

    Buoc 3: Dinh nghia rules nhan dang cap (moi dong = 1 cap):
            Moi rule gom 4 truong:
              [Lv 1] Ten cot moi | Dieu kien  | Pattern     | Strip prefix
              [Lv 1] Kho         | Starts with| Kho         | Kho:
              [Lv 2] Vat tu      | Starts with| Vat tu      | Vat tu:

            - Ten cot moi : ten cot se duoc them vao ket qua
            - Dieu kien   : starts_with / contains / ends_with /
                            is_numeric / is_date / regex
            - Pattern     : gia tri de nhan biet dong cha
            - Strip prefix: phan can xoa khi lay gia tri (de trong = giu nguyen)
            - Bam "+ Them cap" neu co nhieu cap hon

            BADGE "Lv 1", "Lv 2"... cho biet thu tu uu tien nhan dang.
            Rule nao o tren duoc uu tien truoc (match rule nao thi thoi).

    Buoc 4: Chon "Dong leaf" (cach nhan biet dong du lieu that):
              La so hoac ngay (date/number) <- pho bien nhat
              Khong khop rule nao
              Khong trong (not empty)

    Buoc 5: Bam "Preview" kiem tra ket qua
            - Cot moi them duoc to mau xanh la
            - Status hien "X dong ket qua, xoa Y dong, them Z cot: [Kho, Vat tu]"

    Buoc 6: Bam "Apply"

  KET QUA:
    Kho          | Vat tu    | Ngay viet  | SCT-Nhap      | Dien giai
    750-EVT-...  | 1.01...   | 2025-01-07 | 01.750.01.001 | Nhap kho
    750-EVT-...  | 1.01...   | 2025-01-08 | (trong)       | Dieu chinh

  DIEU KIEN HO TRO:
    starts_with  : bat dau bang chu "Kho", "Vat tu", "Chuong"...
    contains     : chua chuoi bat ky
    ends_with    : ket thuc bang
    is_numeric   : la so (nhan ra hang so luong, gia tri)
    is_date      : la ngay thang (nhan ra dong du lieu ngay)
    regex        : bieu thuc chinh quy (nang cao)

6. CAST TYPE (DOI KIEU DU LIEU)
=================================

  Badge tren header cot:
    [123] xanh la  = Numeric (so)
    [ABC] xanh navy = Text (chu)
    [DT]  xanh teal = Date (ngay thang)

  Cach doi kieu - chuot phai vao header cot:
    - Cast to Number  (123): chuyen chuoi -> so de tinh toan
    - Cast to Text    (ABC): chuyen so -> chuoi
    - Convert to Date (DT) : chuyen chuoi -> datetime (dd/mm/yyyy)

  Khi nao can:
    - Cot so bi doc la chu -> Sum/Mean bi loi -> Cast to Number truoc
    - Muon giu so 0 dang truoc (ma don hang, so dien thoai) -> Cast to Text
    - Cot ngay bi doc la chu -> sort / tinh khoang cach ngay sai -> Cast to Date

  Auto-detect ngay:
    Phan mem tu dong hien goi y "DT" neu cot co >= 80% gia tri dang dd/mm/yyyy
    Bam goi y de chuyen sang kieu Date ngay.

10. LUU Y VE SO LIEU VIET NAM
==============================

  Dau thap phan:
    - Viet Nam dung phay:  1.500,75  (1500 phay 75)
    - Quoc te dung cham:   1,500.75  (1500 cham 75)
    - Khi mo CSV chon dung "Dau thap phan" de phan mem doc dung

  So co dau cham ngan (vi du: 1.500):
    - Neu decimal=phay -> 1.500 duoc hieu la 1500 (nghin)
    - Neu decimal=cham -> 1.500 duoc hieu la 1.5

  Hien thi:
    - Phan mem tu dong sua "1500.0" -> "1.500" (dang Viet Nam)
    - Badge "123" tren header neu cot la so, "ABC" neu la chu, "DT" neu la ngay

11. AI SEMANTIC FILTER (TIM KIEM GAN DUNG BANG BGE-M3)
=======================================================

  Tim cac dong co NOI DUNG TUONG DONG voi tu khoa,
  KHONG phai tim chinh xac nhu Filter thuong.

  Vi du: Tim "may bom nuoc" ra ca:
    "Centrifugal pump 3HP", "EBARA pump 2 inch", "May bom ly tam",
    "Water pump EBARA", "Bom nuoc cong nghiep"...

  CACH DUNG:
    1. Bam "AI Filter" hoac Transform > AI Semantic Filter...
    2. Chon cot can tim kiem (ten hang hoa, mo ta...)
    3. Nhap noi dung tim kiem
    4. Bam "Tim kiem" -> doi 10-30 giay
       * Lan dau: ~15 giay de nap model BGE-M3 vao RAM (~3-4 GB)
    5. Kiem tra ket qua:
         In dam = se GIUM LAI (diem >= nguong)
         In xam = se BI LOAI  (diem < nguong)
    6. Keo SLIDER "Nguong" de dieu chinh (0.85+ = rat tin tuong)
    7. Bam "Apply" -> ghi vao pipeline (mau tim)

  LUU Y:
    - Day la tim kiem GAN DUNG, khong dam bao 100% chinh xac
    - User PHAI tu kiem tra truoc khi Apply
    - Score >= 0.85: rat tin tuong | 0.70-0.84: can xem lai | < 0.70: thuong sai

12. AI FUZZY DEDUP (GOP TRUNG LAP MO)
======================================

  Xoa cac dong "gan giong nhau" (khac mot it: viet tat, loi chinh ta, dau cach...)
  Khac Drop Duplicates (chi xoa giong 100%).

  Vi du:
    "May bom nuoc EBARA"  ->  giu lai (canonical)
    "May bom EBARA"       ->  xoa (trung lap)
    "Bom EBARA nuoc"      ->  xoa (trung lap)

  CACH DUNG:
    1. Bam "AI Dedup" hoac Transform > AI Fuzzy Dedup...
    2. Chon cot can dedup
    3. Chinh nguong tuong dong (mac dinh 0.85)
       - Cao hon (0.90+): chi xoa khi rat giong
       - Thap hon (0.80-): xoa ca khi hoi khac
    4. Bam "Tim trung lap" -> xem bang cac nhom
       - Cot "Giu lai": gia tri dau tien trong nhom (canonical)
       - Cot "Trung voi": gia tri se bi xoa
       - Score: muc do tuong dong
    5. Bam "Apply" -> chi giu 1 dong dau tien moi nhom

  LUU Y:
    - Nen xem ki bang truoc khi Apply
    - Thay doi nguong -> bam "Tim trung lap" lai de cap nhat

13. AI CROSS-FILE MATCH (KHOP DU LIEU 2 FILE)
==============================================

  Khop tung dong cua Frame A voi gia tri gan nhat trong Frame B
  (tuong tu VLOOKUP nhung dung AI thay vi khop chinh xac).

  Ket qua: Frame A + 3 cot moi:
    _best_match  : gia tri khop tot nhat tu Frame B
    _match_score : diem tuong dong (0.0 - 1.0)
    _match_status: su do muc do (xanh >= 0.80 / vang 0.50-0.79 / do < 0.50)

  CACH DUNG:
    Yeu cau: Can co it nhat 1 Stored Frame (Frame B - lam tham chieu).
    De tao Stored Frame: mo file thu 2, chon Stored Frames panel > luu.

    1. Bam "AI Match" hoac Transform > AI Cross-file Match...
    2. Chon Frame A (hien tai hoac Stored Frame)
    3. Chon Frame B (phai la Stored Frame)
    4. Chon cot khoa cua tung frame
    5. Bam "Match" -> doi ket qua
    6. Ket qua luu vao Stored Frames (khong anh huong pipeline chinh)

  DUNG KHI NAO:
    - Can khop ten hang hoa giua 2 danh muc khac nhau
    - VLOOKUP bi loi vi ten viet tat / khac nhau mot phan
    - Doi chieu du lieu giua 2 he thong
